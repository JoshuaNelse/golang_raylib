# Silly 2D Game in Golang

---

## To run
- `go mod tidy`
- `go run .`

## Controls
- movement: w,a,s,d or arrow keys
- attack: left mouse click
- switch weapons (number keys)
    - 1 regular sword
    - 2 regular bow
    - 3 bow that shoots swords
    - 4 honorary sword that shoots bows
- pause/start music: q
- turn debug mode on/off:
